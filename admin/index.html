📋 Panduan Cepat

  1. Pilih menu di sidebar kiri (misal: Info Kontak)
  2. Isi atau ubah data yang diinginkan
  3. Klik "Simpan Semua" di pojok kanan atas
  4. Buka website utama — data akan otomatis terupdate
Data disimpan secara lokal di browser menggunakan localStorage. Data akan tetap ada meski browser ditutup dan dibuka kembali.
